# simple-test-client

## Description
Simple TCP-client  
default server ip: "0.0.0.0"  
default server port: 8080  
This client used for test this hash-tcp-server: https://github.com/sergeya3ro/simple-hash-server  

## Install
```python
chmod +x build_client.sh
./build_client.sh
```

## Run client
```python
cd build/bin
./client
```
